# Internet-Draft Repository

The AID Internet-Draft source has moved to a dedicated repository following standard IETF practice.

**Repo:** `agentcommunity/draft-nemethi-aid-agent-identity-discovery`

**Editor's Copy (when public):**
https://agentcommunity.github.io/draft-nemethi-aid-agent-identity-discovery/draft-nemethi-aid-agent-identity-discovery.html

**Datatracker (after submission):**
https://datatracker.ietf.org/doc/draft-nemethi-aid-agent-identity-discovery/

The I-D repo uses [martinthomson/i-d-template](https://github.com/martinthomson/i-d-template) for automated builds and GitHub Pages publishing. The XML source uses the `-latest` naming convention; version numbers are assigned automatically on tagged releases.

## What stays here

This `tracking/iana/` directory retains operational and evidence files for the IANA registration process:

- `EVIDENCE_ANNEX.md` — reproducible evidence for reviewer questions
- `AID_DEVELOPMENT_TIMELINE.md` — protocol development history
- `IANA-ICANN-Research-Report.md` — research on IANA/ICANN processes
- `AGENTS.md` — agent instructions for this directory
